# steam-cli

🇷🇺 Русский · [🇬🇧 English](./README.md)

**Реальные данные из Steam — прямо в руки вашему AI-агенту.** Отзывы и их
тональность, информация о магазине (жанры, дата выхода, разработчики, цена,
Metacritic), онлайн игроков сейчас, патчноуты, проценты ачивок, распродажи,
а также рыночная разведка: пользовательские теги, размер ниши, похожие игры,
история отзывов и грубая оценка продаж.

Это однофайловая утилита на Python (только стандартная библиотека, **без
зависимостей**), и для неё **не нужен ключ Steam Web API** — всё берётся из
публичных эндпоинтов Steam.

---

## Зачем это инди-разработчику и издателю

Steam-страница, отзывы, теги и распродажи — это открытые данные, но собирать
их вручную по сотням игр невозможно. `steam-cli` достаёт их одной командой, а
главное — отдаёт в виде, который **понимает AI-агент**. На практике это значит,
что вы можете просто попросить своего агента:

- «Как приняли *Stardew Valley*? Дай оценку и главные жалобы из негативных
  отзывов» → агент сам сходит за отзывами и пересоберёт настроение аудитории.
- «Насколько забита ниша cozy-фермеров до 15$? Покажи размер и список» →
  агент посчитает, сколько игр в нише, и выдаст конкурентов.
- «Прикинь, сколько копий продала вот эта игра» → грубая оценка владельцев и
  выручки по числу отзывов (метод Boxleiter).
- «Найди игры, похожие на мою, для аутрич-списка» → набор соседних тайтлов.
- «Удержался ли лонч этой игры или просел?» → история скорости отзывов во
  времени.
- «Что сейчас в распродаже / в топе продаж?», «Сколько онлайна у конкурента?»,
  «Под какими тегами выходит эта игра?» — и так далее.

Коротко: это инструмент **рыночной разведки по Steam** для тех, кто делает или
издаёт игры.

---

## Главная идея: утилита написана под AI-агентов

`steam-cli` можно запускать руками в терминале, но придумана она для другого
сценария — **чтобы за вас её вызывал AI-агент** (Claude Code, Codex CLI,
opencode и любые другие). Вы пишете агенту задачу на обычном языке, а он сам
выбирает нужные команды и разбирает ответ.

Поэтому вся утилита спроектирована под машинное потребление:

- у **каждой** команды есть флаг `--json` — структурированный ответ, который
  агент парсит без догадок;
- **ошибки тоже в JSON** (`{"error": ..., "code": ...}`) — агент всегда
  получает валидный объект, а не пустой вывод;
- служебные пометки идут в `stderr`, чтобы не пачкать данные в `stdout`;
- к проекту приложены готовые инструкции для агентов — скилл для Claude Code и
  файл `AGENTS.md` для Codex/opencode (см. раздел [«Для AI-агентов»](#для-ai-агентов)).

Если вы работаете с играми через AI-ассистента — это «руки», которыми он
дотягивается до живого Steam.

## Установка

> Аудитория этого репозитория — в том числе не-технические люди. Ниже всё
> расписано по шагам; если что-то не сработало, загляните в раздел
> [«Если команда не находится»](#если-команда-steam-cli-не-находится).

### Шаг 1. Проверьте, что есть Python 3.8 или новее

Утилита работает на Windows, macOS и Linux, но ей нужен установленный Python.
Откройте терминал (на Windows — **PowerShell**, на macOS — **Terminal**) и
введите:

```bash
python3 --version
```

Если видите что-то вроде `Python 3.11.x` — всё в порядке, переходите к шагу 2.
Если команда не найдена — поставьте Python:

- **Windows:** установите из [python.org/downloads](https://www.python.org/downloads/)
  и **обязательно** отметьте галочку **«Add Python to PATH»** в начале
  установщика. Или одной командой в PowerShell: `winget install Python.Python.3.12`.
- **macOS:** `brew install python` (если установлен [Homebrew](https://brew.sh)),
  либо скачайте с [python.org](https://www.python.org/downloads/).
- **Linux:** обычно уже стоит; иначе `sudo apt install python3 python3-pip`
  (Debian/Ubuntu) или аналог для вашего дистрибутива.

### Шаг 2. Установите steam-cli одной командой

```bash
pip install git+https://github.com/dim-s/steam-cli.git
```

Эта команда создаёт программу `steam-cli`, доступную из любой папки. Чтобы
позже обновиться до новой версии, добавьте `-U`:
`pip install -U git+https://github.com/dim-s/steam-cli.git`.

> **На Windows** команда та же — просто выполните её в PowerShell. Если `pip`
> не найден, попробуйте `python -m pip install git+https://github.com/dim-s/steam-cli.git`.

### Шаг 3. Проверьте установку

```bash
steam-cli --version       # → steam-cli 1.3.1
steam-cli overview "Hades"
```

Если показалась версия и краткая карточка игры — готово. 🎉

### Если команда `steam-cli` не находится

Скорее всего, папка, куда `pip` кладёт программы, не добавлена в `PATH`:

- **Windows:** это `%APPDATA%\Python\PythonXY\Scripts`. Проще всего —
  переустановить Python с галочкой «Add Python to PATH», либо запускать через
  `python -m steam_cli ...`.
- **macOS / Linux:** это `~/.local/bin`. Добавьте строку
  `export PATH="$HOME/.local/bin:$PATH"` в `~/.zshrc` (macOS) или `~/.bashrc`
  (Linux) и перезапустите терминал.

### Альтернативы для продвинутых

```bash
# pipx — ставит в изолированное окружение (не мешает другим Python-проектам)
pipx install git+https://github.com/dim-s/steam-cli.git

# uv tool — самый быстрый способ, если установлен uv
uv tool install git+https://github.com/dim-s/steam-cli.git

# совсем без установки — разово запустить через uv
uvx --from git+https://github.com/dim-s/steam-cli.git steam-cli reviews "Hades" --summary
```

## Для AI-агентов

Главный сценарий: агент сам вызывает `steam-cli`, когда вашей задаче нужны
живые данные Steam. Ниже — как подключить разные агенты.

### Claude Code

В папке [`skill/`](./skill) лежит готовый скилл (`SKILL.md` + `setup.md`):
агент сам поймёт, когда его применить, вызовет `steam-cli` и при отсутствии —
сам себя доустановит. Подключается симлинком в вашу папку скиллов:

```bash
ln -s "$(pwd)/skill" ~/.claude/skills/steam
```

Затем перезапустите Claude Code (или `/reload`), чтобы скилл проиндексировался.
После этого достаточно писать обычные запросы — «посмотри отзывы X на Steam»,
«насколько велика ниша cozy-игр» — и агент сам сходит за данными.

### Codex CLI, opencode и другие агенты с AGENTS.md

Эти агенты читают файл `AGENTS.md` из корня проекта. В этом репозитории такой
файл уже есть — если вы клонировали steam-cli и работаете внутри него, агент
подхватит контракт автоматически.

Чтобы агент знал про `steam-cli` **в вашем собственном проекте** (например, в
репозитории вашей игры), добавьте в его `AGENTS.md` короткий блок:

```markdown
## Данные Steam (steam-cli)

`steam-cli` установлен и даёт реальные данные Steam без API-ключа.
Используй его для рыночной разведки и всегда добавляй `--json` при разборе.

- steam-cli overview "<игра>" --json            # сводка: инфо + оценка + онлайн + цена
- steam-cli reviews "<игра>" --summary --json   # тональность отзывов
- steam-cli browse --tags cozy,roguelike --max-price 15 --json   # размер ниши
- steam-cli similar "<игра>" --json             # похожие игры
- steam-cli overview "<игра>" --estimate --json # грубая оценка продаж
```

### Любой другой агент

Подойдёт любой агент, умеющий запускать команды в терминале и читать JSON.
Достаточно установить утилиту (см. выше) и вызывать команды с `--json`:

```bash
steam-cli overview "Hollow Knight" --json
```

Контракт, на который опирается агент:

- **`--json` есть у каждой команды** — машиночитаемый вывод в `stdout`;
- **ошибки — тоже JSON** в `stdout` (`{"error": ..., "code": ...}`) с ненулевым
  кодом возврата, парсер всегда получает объект;
- **служебная пометка о выборе appid идёт в `stderr`** — `stdout` остаётся
  чистым (отключается флагом `-q`);
- **языки и регионы нормализуются** (`--lang ru`, `--cc usa`→`us`), неизвестный
  код — явная ошибка, а не молча неверные данные;
- **кэш и повторы запросов** — автоматические; временные сбои сети повторяются.

## Быстрый старт (вручную в терминале)

Везде `<игра>` — это **appid** (`1145360`) или **название** (`"Hades"`,
разрешается через поиск; выбранный appid печатается в `stderr`).

```bash
# Сводка одним запросом: инфо + оценка отзывов + онлайн + цена
steam-cli overview "Hades"

# Быстрая тональность: оценка и количество отзывов, без скачивания текстов
steam-cli reviews "Hades" --summary

# 200 свежих отзывов на английском в JSON — для разбора агентом
steam-cli reviews 1145360 -n 200 --language english --json

# Только негативные отзывы — найти жалобы
steam-cli reviews 1145360 --review-type negative -n 100 --json

# Карточка магазина: жанры, дата выхода, разработчики, Metacritic, цена
steam-cli info "Baldur's Gate 3"

# Рыночная разведка: теги, размер ниши, похожие, история отзывов, оценка продаж
steam-cli tags "Hades"                                 # теги с весом голосов
steam-cli browse --tags cozy,roguelike --max-price 15  # размер и список ниши
steam-cli similar "Hades"                              # «ещё похожие»
steam-cli history "Hades"                              # всплеск на лонче vs сейчас
steam-cli overview "Hades" --estimate                  # грубая оценка владельцев/выручки

# Сравнить цену по регионам
steam-cli price 1145360 --cc us,de,ru,br

# Онлайн игроков сейчас, новости/патчноуты, проценты ачивок
steam-cli players 570
steam-cli news 1145360 -c 5
steam-cli achievements 1145360

# Что в распродаже и что продаётся прямо сейчас
steam-cli specials
steam-cli top-sellers

# Скачать обложку и скриншоты, чтобы агент мог *увидеть* игру
steam-cli images 1145360 --what header screenshots --limit 4 --out ./media
```

## Команды

| Команда | Что делает |
|---|---|
| `overview <игра>` | Сводка одним запросом: инфо + оценка + онлайн + цена |
| `reviews <игра>` | Пользовательские отзывы + агрегированная оценка |
| `info <игра>` | Карточка магазина: жанры, дата, разработчики, Metacritic, цена |
| `images <игра>` | Скачать обложку / скриншоты / арт в файлы |
| `search <запрос>` | Название → кандидаты appid |
| `players <игра>` | Онлайн игроков сейчас |
| `news <игра>` | Новости / патчноуты |
| `achievements <игра>` | Глобальные проценты выполнения ачивок |
| `price <игра>` | Цена и скидка по одному или нескольким регионам |
| `specials` | Игры со скидкой прямо сейчас |
| `top-sellers` | Текущие топ-продажи |
| `tags <игра>` | Пользовательские теги с числом голосов (Cozy, Roguelike…) |
| `browse --tags …` | Поиск по нише: её размер и список по тегу/цене/сортировке |
| `similar <игра>` | Набор рекомендаций «ещё похожие» |
| `history <игра>` | Скорость отзывов во времени (всплеск на лонче vs сейчас) |
| `profile <id>` | Публичный профиль Steam Community (без ключа) |
| `cache [--path/--clear]` | Посмотреть или очистить локальный кэш |

Добавьте `--json` к любой команде для структурированного вывода. Полный
справочник по флагам и форматам ответов — в [`skill/SKILL.md`](./skill/SKILL.md),
а английская версия README — [здесь](./README.md).

## Ограничения (чтобы не было сюрпризов)

- **Без API-ключа** — используются только публичные эндпоинты Steam.
  Библиотека игр конкретного пользователя, приватные профили и персональные
  ачивки требуют ключа Steam Web API и сознательно не поддерживаются.
- **Разрешение имени берёт первый результат поиска** — `"Hades"` может
  разрешиться в *Hades II*. Когда важна точность, передайте appid или сверьтесь
  через `steam-cli search`; выбранный appid всегда печатается в `stderr`.
- **`overview --estimate` — это эвристика Boxleiter** (владельцы ≈ отзывы ×
  множитель), выдаётся диапазоном. Только порядок величины, не точная цифра
  Steam.
- **Вишлисты, подписчики и исторические кривые онлайна/цены недоступны** без
  ключа — они живут только в SteamDB или партнёрском бэкенде Steamworks.
  `history` (скорость отзывов) — ближайший публичный заменитель для оценки
  динамики.
